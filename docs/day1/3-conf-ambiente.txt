Comandos importantes:

python --version = python -V: exibe a versão

python -m - para utilizar módulos do python, como python -m turtledemo, por exemplo

python -VV = exibe mais informações sobre o python

python -h - exibe opções e argumentos para utilização do python em linha de comando; é igual a python --help

python -c ["comando"] - executa um comando

python -m site - caminhos onde o python busca os pacotes, ou seja, o sys.path, onde o python está instalado.

Python interpreter on command line - REPL - Read, Eval, Print, Loop - ler, interpretar, imprimir, voltar para o loop

Ao utilizar o intepretador na linha de comando, o comando

>>> help() ou >>> help(objeto) para obter ajuda acerca da própria linguagem. É uma ótima fonte pra linguagem.

Ao utilizar help(), abrirá uma linha de comando específica de ajuda, com a seguinte carinha:

help>

Aqui, é possível pesquisar qualquer coisa, como, por exemplo:

help> modules

que vai exibir todos os módulos de python instalados no computador

help>> topic - exibe uma quantidade imensa de tópicos relacionados à programação
